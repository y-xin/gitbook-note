# 安装

安装GitBook非常简单。

**要求**

* NodeJS（建议使用v4.0.0及以上版本）
* Windows，Linux，Unix或Mac OS X

**通过NPM安装**

```
$ npm install gitbook-cli -g
```
安装成功
```
$ gitbook -V
CLI version: 2.3.2
GitBook version: 3.2.3
```
