# GitBook 上手笔记

> GitBook 是一个结合 Github/Git 和 Markdown 可生成静态页面和电子书的软件。
使用gitbook 命令可以生成一下文档
* 静态站点：GitBook默认输出该种格式
* PDF：需要安装gitbook-pdf依赖
* eBook：需要安装ebook-convert

> GitBook Editor 是一个本地应用版的在线编辑器，所见即所得。
